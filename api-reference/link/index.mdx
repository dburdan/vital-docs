# Overview

## Generate Link Token

<Swagger endpoint={"/link/token"} method={"post"} title={"Create a new link token"} include_client_instantiation>

```bash
curl --request POST \
     --url {{BASE_URL}}/v2/link/token \
     --header 'Accept: application/json' \
     --header 'Content-Type: application/json' \
     --header 'x-vital-api-key: <API_KEY>' \
     --data '
{
     "user_id": "1875c190-0cd6-46c1-8670-5b56a7794b78"
}
'
```

```js
const user = await client.Link.create(user_id);
```

```python
user = client.Link.create(user_id)
```

</Swagger>

## Link OAuth provider

<Swagger endpoint={"/link/provider/oauth/{oauth_provider}"} method={"get"} title={"Link an OAuth Provider"} include_client_instantiation>

```bash
curl --request GET \
     --url {{BASE_URL}}/v2/link/provider/oauth/{oauth_provider} \
     --header 'Accept: application/json'
     --header 'Content-Type: application/json' \
     --header 'x-vital-link-token: <VITAL-LINK-TOKEN>'
```

```js
const user = await client.Link.getOAuthLink(provider);
```

```python
user = client.Link.oauth_provider(provider)
```

</Swagger>

## Link Password Provider

<Swagger endpoint={"/link/provider/password/{provider}"} method={"post"} title={"Link a password provider"} include_client_instantiation>

```bash
curl --request POST \
     --url {{BASE_URL}}/v2/link/provider/password/{provider} \
     --header 'Accept: application/json' \
     --header 'Content-Type: application/json' \
     --header 'x-vital-link-token: <VITAL-LINK-TOKEN>'
     --data '
{
     "username": "test",
     "password": "test"
}
'
```

```js
const user = await client.Link.connectProvider(
  linkToken,
  provider,
  username,
  password
);
```

```python
user = client.Link.password_provider(link_token, provider, username, password)
```

</Swagger>
