# Cholesterol

<Swagger endpoint={"/timeseries/{user_id}/cholesterol/total"} method={"get"} title={"Get total cholesterol data"} include_client_instantiation>

```bash
curl --request GET \
     --url {{BASE_URL}}/v2/timeseries/{user_id}/cholesterol/hdl?start_date=2021-01-01&end_date=2021-02-01 \
     --header 'Accept: application/json'
     --header 'x-vital-api-key: <API_KEY>'
```

```python
data = client.Vitals.cholesterol(
        type="total",
        user_id=*,
        start_date="2020-01-01",
        end_date="2021-01-01"
        )
```

</Swagger>
