When you sign up to Vital you get access to two envrionments, Sandbox and Production.

| Environment |
| ----------- | ------------------------------- |
| Production  | https://api.tryvital.io         |
| Sandbox     | https://api.sandbox.tryvital.io |

## Sandbox

Our sandbox environment:

- is for app integration
- is for formal integration testing
- up to 20 live devices
- stateful, data is persisted
- requires authentication

## Production

Our production environment:

- uses production data
- should not be used for testing
- unlimited devices
