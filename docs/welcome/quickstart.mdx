## Step 1: API keys

Start by creating an account at https://app.tryvital.io. Once registered, you can create a team. A team is associated with a region (either EU, or US), please select the one most suitable for you. Finally, you will have access to API Keys for both Sandbox and Production.

:::info
If you get stuck at any point in the Quickstart, help is just a click away! Join our Slack channel or send us a message to support@tryvital.io
:::

## Step 2: Creating Your First User

You create a _user_ with the `/user/` endpoint. Once done, you will get an `user_id`. Most of our API requests require an `user_id` as input. When creating a user, you need to pass an unique id. This represents the user in your system.

:::caution
Personally identifiable information (PII), such as an email address or phone number, should not be used as input for the `user_id` parameter.
:::

<br />
<br />

<CodeBlock title={"Creating a Vital user"} include_client_instantiation={true}>

```bash
curl --request POST \
     --url {{BASE_URL}}/v2/user/ \
     --header 'Accept: application/json' \
     --header 'Content-Type: application/json' \
     --header 'x-vital-api-key: <your_api_key>' \
     --data '{"client_user_id":"<your_user_id>"}'
```

```js
const user = await client.User.create(client_user_id);
```

```python

user = client.User.create(client_user_id)
```

```swift
let user = try await VitalNetworkClient.shared.user.create(clientUserId)
```

</CodeBlock>

## Step 3: Making your first API request

:::tip
You can use our [Quickstart project](https://github.com/tryVital/quickstart) with a complete backend and frontend for you to get a feeling of our API. You can also try our [iOS demo](https://github.com/tryVital/vital-ios) to explore different ways of interacting with our services.
:::

For our first API request, we will call `/summary/sleep`:

<br />

<CodeBlock title={"Getting User sleep data"} include_client_instantiation={true}>

```python
sleep_data = client.Sleep.get(user_id, start_date, end_date)
```

```js
const sleepData = await client.Sleep.get(userId, startDate, endDate);
```

```swift
let sleepData = try await VitalNetworkClient.shared.summary.sleep(userId, startDate, endDate)
```

</CodeBlock>
