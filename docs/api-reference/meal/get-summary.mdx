<Info>
All values in the returned model are returned in the following units:
</Info>

- `meals[*].energy.value` in calories `kcal`
- `meals[*].macros.carbs` in grams `g`
- `meals[*].macros.protein` in grams `g`
- `meals[*].macros.fats.saturated` in grams `g`
- `meals[*].macros.fats.monounsaturated` in grams `g`
- `meals[*].macros.fats.polyunsaturated` in grams `g`
- `meals[*].macros.fats.omega3` in grams `g`
- `meals[*].macros.fats.omega6` in grams `g`
- `meals[*].macros.alcohol` in litres `lt`
- `meals[*].macros.water` in litres `lt`
- `meals[*].macros.fibre` in grams `g`
- `meals[*].macros.sugar` in grams `g`
- `meals[*].micros.minerals` key: value pairs in milligrams `mg`
- `meals[*].micros.trace_elements` key: value pairs in milligrams `mg`
- `meals[*].micros.vitamins` key: value pairs in milligrams `mg`
