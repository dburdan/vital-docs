Cancelling an order depends how far along in the process the order is in. If an order is still
in a state where the requisition form has not been generated, order has not been sent out, or
phlebotmist has not been sent that it is possible to make the below request.

<CodeBlock endpoint={"/lab_test/orders/{order_id}/cancel"} method={"post"} title={"Cancelling an order"} include_client_instantiation>

```bash
curl --request DELETE \
     --url {{BASE_URL}}/v3/lab_test/<order_id>/cancel \
     --header 'Accept: application/json' \
     --header 'x-vital-api-key: <API_KEY>' \
     --header 'Content-Type: application/json' \
```

```python
data = client.Testkits.cancel(user_id=*)
```

```js
data = client.Testkits.cancel(user_id=*)
```

</CodeBlock>

You will receive a response that will specify whether the order was able to be cancelled or not.
